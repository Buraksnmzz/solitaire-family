Changelog — Version 2.0.0
Date: 2025-06-03

Changes & Improvements:
-Updated AdjustManager to initialize even when ATT is denied, ensuring ad revenue tracking continues correctly.

-Removed deprecated Amazon Ads integration from PluginAdsManagerController — AppLovin is now the sole ad network.

-Fixed IAP receipt handling logic to ensure No Ads purchase is correctly recognized after restore and install.

-Unified session tracking logic in PluginSessionManager (switched to use SetString for install time and added const keys).

-Improved Crashlytics logging by replacing Crashlytics.Log() with structured LogException() calls for better stack trace reporting.

-Adjusted ad revenue logging to occur only on OnAdRevenuePaidEvent instead of also on OnAdClickedEvent.

-Removed conditional Adjust revenue skipping on ATT denial to align with Adjust’s guidelines.

-General testing and bug fixes related to internal testing purchases, Play Store restore behavior, and first session ad control.

- Added null checks for parameters in YoogoLabManager.LogEvent()
